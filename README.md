# AI 知识库 📚🤖

基于 Spring AI + Vue 3 的个人知识库系统，支持文档上传、语义检索和 AI 问答。

## 🔆 项目定位

个人学习资料管理工具，通过 RAG（检索增强生成）技术，让 AI 基于你的文档回答问题。

## 🚀 核心功能

- 📤 **文档上传** - 支持 PDF/Word/Excel/Markdown 等多种格式
- 🔍 **语义检索** - 基于向量相似度搜索
- 💬 **AI 问答** - 基于知识库内容的智能回答
- 📁 **分类管理** - 文档分类/标签组织
- 👤 **用户系统** - 登录/注册/权限管理

## 🛠️ 技术栈

### 后端
- Spring Boot 3
- Spring AI
- PostgreSQL + pgvector
- Redis
- Spring Security + JWT

### 前端
- Vue 3 + Vite + TypeScript
- Element Plus
- Pinia
- Axios

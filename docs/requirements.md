# 需求文档 📋

## 1. 项目概述

### 1.1 项目名称
AI 知识库 (AI Knowledge Base)

### 1.2 项目类型
个人项目 - 学习资料管理

### 1.3 目标用户
个人使用（Tang）

### 1.4 核心价值
通过 RAG 技术，让 AI 基于个人学习资料回答问题，提高学习效率。

---

## 2. 功能需求

### 2.1 用户系统 🔐

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 注册 | 邮箱/用户名 + 密码注册 | P0 |
| 登录 | JWT Token 认证 | P0 |
| 登出 | Token 失效（Redis 黑名单） | P0 |
| 个人信息 | 查看/修改基本信息 | P1 |

### 2.2 知识库管理 📚

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建知识库 | 单个知识库（个人使用） | P0 |
| 分类管理 | 创建/编辑/删除分类 | P0 |
| 标签管理 | 添加/移除标签 | P1 |

### 2.3 文档管理 📄

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 上传文档 | 支持拖拽/批量上传 | P0 |
| 文档解析 | PDF/Word/Excel/Markdown/TXT | P0 |
| 文档列表 | 查看/搜索/筛选 | P0 |
| 文档删除 | 软删除（可恢复） | P1 |
| 文档预览 | 在线预览内容 | P2 |

### 2.4 AI 问答 💬

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 智能问答 | 基于知识库内容回答 | P0 |
| 流式输出 | 打字机效果 | P0 |
| 历史记录 | 保存对话历史 | P0 |
| 上下文理解 | 多轮对话支持 | P1 |
| 引用来源 | 显示答案来源文档 | P1 |

### 2.5 系统功能 ⚙️

| 功能 | 描述 | 优先级 |
|------|------|--------|
| Redis 缓存 | 查询结果/会话缓存 | P0 |
| 限流 | 防止 API 滥用 | P1 |
| 日志 | 操作日志/错误日志 | P1 |

---

## 3. 非功能需求

### 3.1 性能
- 问答响应时间 < 3 秒
- 文档上传支持 100MB 以内
- 并发用户数：1-5 人（个人使用）

### 3.2 安全
- 密码加密存储（BCrypt）
- JWT Token 认证
- SQL 注入防护
- XSS 防护

### 3.3 可用性
- 支持本地运行
- 支持服务器部署
- 数据可备份/恢复

---

## 4. 技术约束

- 后端：Java 17+
- 前端：现代浏览器
- 数据库：PostgreSQL 14+
- 缓存：Redis 6+
- AI 服务：DeepSeek API
